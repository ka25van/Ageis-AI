# Screenshots & Demo

## Required Screenshots

Capture these pages in order:

1. **Dashboard** — `dashboard.png` — shows project/repo/run counts
2. **Projects** — `projects.png` — shows project list with create button
3. **Repositories** — `repositories.png` — shows indexed repo with status
4. **Knowledge** — `knowledge.png` — shows search results with chunks
5. **Agents** — `agents.png` — shows agent cards with a result displayed
6. **Settings** — `settings.png` — shows profile/API key sections

## Instructions

1. Start backend + frontend
2. Create a project, connect a GitHub repo, index it
3. Run the Repository Agent to show a result
4. Take screenshots (Windows: `Win + Shift + S`, macOS: `Cmd + Shift + 4`)
5. Save as PNG in this directory with the names above

## Demo GIF

Use a free tool like [ScreenToGif](https://www.screentogif.com/) (Windows) or [Kap](https://getkap.co/) (macOS):

1. Record a 30-60 second walkthrough showing:
   - Creating a project
   - Connecting a repository
   - Running the Repository Agent
   - Showing the result
2. Export as `demo.gif` (keep under 5MB)
3. Save to this directory
